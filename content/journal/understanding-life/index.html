感受生活{初中时代的作文}

August 21 2006

感受生活 \ 生活就像五味瓶，酸甜苦辣才叫人生。\ \ 苦 \ 2002年暑假，悲痛和迷茫在我心中挥之不散。\ \

我的小学生活即将结束，父母正忙于为我寻找一所好初中。这时，外公传来噩耗。我们全家马上坐车去外公家。车上，父亲还在打电话为我联系升初中的事。我的心情很沉重，好像突然间失去了很多东西。\ \

到了外公家，我看见外公躺在床上奄奄一息的样子，简直说不出话来。外婆告诉外公我来了，我不知道他听见没有，不知道他看见没有。我为外公做的最后一件事，就是用吸管喂他喝水。外公出殡的那一天，我看着外公的灵柩被运上车，抱着爸爸哭了，那一刻，我希望自己一直不相信的天堂真的存在，而外公就在上面凝视着我。\ \

刚好那一天，是小学的毕业典礼，我对于小学最后一页的记忆，是空白的，就好像始终没有毕业。\ \ 辣\ 2004年校运会，在火辣辣的太阳下，我们有一颗炽热的心。\ \

三级跳是初三级的新项目。校运会前1个月，我们运动员几乎每天下午都到中山医体育场训练。由于有很好的跳远基础和老师指导，胡乔宪进步得很快，我们都对她抱很大希望。\ \

谁知，天有不测之风云。跳远时，胡乔宪的动作就像三级跳一样，起跳时离踏板还有几十厘米！成绩当然令人大跌眼镜。胡乔宪一脸茫然地坐在跑道边。最后，胡乔宪三甲不入。\ \

到了三级跳，我只有不断地鼓励胡乔宪，不断地对她说要放松，不断地帮她松弛肌肉。虽然她没有说什么，但我知道，在众人期待的目光下，她顶着很大的压力。第一跳只有8米多，2班的余莹已经跳了9米。现在每一跳都是关键。胡乔宪起跑了，我大喊道：“放松！放松！”随即周围爆发出掌声，9米4！胡乔宪欣喜若狂地跑过来拥抱着我，激动得不知道说什么。那一刻，我感觉到正因为我们拥有一颗炽热的心，才能战胜所有困难，最终发光发热。\ \ 酸\ \ 2004年某天早晨，在上学的路上，我感到心酸。\ \

拐了弯就到学校了，这时，一个小男孩突然走出来拦住我。我看看旁边，原来还有一个中年妇女坐着，在她面前放着一个不锈钢碗。我立刻明白了，小男孩缠着我要钱。这种乞丐伎俩早已为人们憎恶。我准备避开他走，可他却挡住我去路，我开始对他产生厌恶，低头望望他的脏衣服，生怕他在我的衣服上抹几点黑。可当我注视着他的眼睛，在他的黑脸蛋儿上显得那么闪亮，那么真诚。我突然感到一种说不出的难过。小小年纪就要这样生活，无论真假，何人不起怜爱之情？\

可我真的没带钱，只好做了个“不”的手势，而他也终于“放过”我了。这种令人心酸的乞丐何时才会消失？\ \ 甜\ \ 2003年某天，我真正感觉到什么叫“像吃了蜜一样甜”。\ \

钟伽文跟我讲起开学初的一件事，说自从那件事后，她就觉得，我是一个乐于帮助别人的好人。这使我很吃惊，也很高兴。\ \

那天发校服，在回家的路上，我看见一个我们学校的学生在士多店前张望，好像很苦恼的样子。我隐约记得她是我的新同学，于是走上前，问是否可以帮助她。原来，她那绑校服的绳子断了，手上又拿着很多东西，不知道怎么把它们拿回家。幸好，士多店的老板有绳子，我帮她绑好了校服，然后就回家了。\ \

若不是钟伽文提起，我都差点忘掉这件事了。没想到，一次小小的普通的帮助可以影响一个人对我的看法。如果人们能够互相帮助，那么冲突就会少很多，我们的生活就更加和谐。\ \ 生活就像五味瓶，酸甜苦辣才叫人生。 Created by X&Y
