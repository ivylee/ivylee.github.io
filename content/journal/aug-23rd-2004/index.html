2004年8月23日{初中时代的暑假日记}

August 21 2006

2004年8月23日 晴 \

今天，原计划舅舅和他的同事从深圳来到广州的广东美术馆看国际著名设计师靳埭强的设计展，舅舅也约了我一起去。\ \

我和妈妈吃完午饭后，就来到广东美术馆。我们先看了这个设计展的宣传海报，它的全称是“生活·心源——靳埭强设计与艺术展”；再看看时间，我们正好赶上了配合此展览的第二个讲座——靳埭强的设计与艺术，于是，舅舅把我们带到了报告厅。由于舅舅所在的公司正是靳埭强先生在深圳开设的分公司，因此我有幸坐在第一排，近距离地听大师演讲。\ \

在我们入场后，陆续又有许多人进场，一部分人甚至站在场边或坐在地板上，可见靳埭强这个名字有多大吸引力。\

讲座分几部分进行。靳先生讲述了他如何从一个小裁缝转变成一位设计师，以及中西文化对他创作的影响等等，同时他也是一位水墨画家，他常常将水墨融入到设计中，别树一帜。\ \

最特别的就是一个红点。起初是他的水墨画老师用以代表荷花，后来靳先生赋予了红点多种含义，有“心”、“莲花”、“红日”、“伤口”等，富有新意。\ \

虽然有些作品我不大明白，但我知道了靳先生认为设计最重要的一点——创意。无论何时，创意都是最重要的，有了好的创意，就成功了一半；而创意的来源就是我们身边，生活、社会、文化、大自然，只要细心观察，就会有收获。\

靳先生不仅是一个设计师，而且是一个商人。在最后的问答过程中，靳先生回答设计师提出的客户不满设计成果的问题，主要以与客户沟通为主；从设计一开始就与客户商量，了解产品、做市场调查等等。许多设计师只讲艺术，认为自己设计得很好，不配合产品市场，而靳先生不然，这也是他能在商业上成功的原因之一。\ \

讲座结束后，我竟有幸和靳先生共进晚餐。舅舅公司的所有成员和我坐车来到离美术馆不远的酒店吃饭，虽然有点紧张，但想起刚才一拥而上排队拿签名的听众，我已是十分幸运了。\ \ 后来，舅舅还帮我拿了一本有靳先生自己亲笔签名的设计作品集。\ \

这个晚上的确很难忘。我竟然与大师如此近距离地接触，这是多少人梦寐以求的，而我却如此简单地与他吃晚饭，这大概就是缘分吧。靳先生说过，能加入他们的公司工作的人都是有缘分的，希望我日后也能有机会向他学习。 Created by X&Y
