2004年7月18日{初中时代的暑假日记}

August 21 2006

2004年7月18日 晴 \

今天下午，我刚睡醒，走到厅里，发现爷爷一个人站在鞋柜边摆弄着，不知道在干什么。\ \

我走上前去，看见鞋柜上有一把大剪刀、一把大尺子、一支红水笔、一把锯刀（像是从什么地方拆下来的，没把手）、几片长方形的塑料片或是铝片，真猜不出爷爷在干什么。于是我问爷爷：“爷爷，这些是塑料片还是铝片？”爷爷回答说：“这些是铝片，看起来是有点像塑料。”“那您在干什么啊？”爷爷微微笑了笑，神秘地说：“呵呵，不告诉你，你肯定猜不着，这东西用处大着呢！”哈哈，爷爷还装神秘！我在他旁边仔细观察观察，我就不信我猜不着！\ \

铝片比较容易定型。只见爷爷量了量尺寸，在铝片长边的两端留了一段距离，用红笔点上两点，然后拿锯刀分别锯开垂直长边、以两个红点为端点的线；接着把分开的三部分往里折，就像做个长方形的盒子一样。我猜道：“爷爷，您在做个盒子吧？”爷爷仍是笑了笑，说：“保密。”“不是盒子是什么？这不是一个无盖的长方体吗？”“嘿嘿，看下去就知道了。”\ \ 我还是第一次看到爷爷这么神秘，这东西究竟是什么呢？有什么用呢？\ \

果然不是盒子！因为爷爷只把2条长和1条宽折起来，这样就装不了东西。那会是什么呢？是用来铲东西的吗？也不是，这个东西容积不大……\ \

正想着，爷爷用剪刀把两端折入的小长方形剪去两个小三角形，再把两边往里压一压。这时，爷爷拿起做好的东西，走到浴室里，在抽风机侧面的空隙比划着。\ \ “哦，原来是这样！”“哈哈，想不到吧？退休了就该干这事！”\ \

上星期浴室装修，抽风机侧面贴不了瓷砖，所以爷爷想用这个东西嵌进去，使浴室看起来美观些。爷爷真是有创意，难怪他说我肯定猜不着！\ \

第一次“试嵌”，这个东西的角大了些，嵌不进去。经过爷爷的“整形”，第二次它就紧紧地嵌了进去，大小、位置刚刚好。看来爷爷对自己的创意也挺自豪，满脸欣喜，一边用俄语说着“好极了！”，一边喊着奶奶来看他的杰作。\ \

我以为把这东西嵌进去就大功告成了，谁知爷爷又拿来一小罐白漆和一支旧毛笔给它上色。\ \

真正大功告成了，爷爷脸上露出了满意的笑容，再一次用俄语赞不绝口“好极了！”“好极了！”……\ \

我真佩服爷爷，对于这样一点小事，都想得如此周到、细致，换了我肯定一早就不耐烦了。爷爷不愧是爷爷！ Created by X&Y
