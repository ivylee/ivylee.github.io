狗{2005年寒假作文}

August 21 2006

狗 \

2005年圣诞节，我和妈妈去逛中华广场。我们漫无目的地上了三楼，走着走着，突然我看见一堆好漂亮的小狗公仔。我拉着妈妈过去，妈妈也惊讶了。好漂亮！好逼真！好可爱！一问售货员，才知道这些是进口的，只在圣诞节时卖。妈妈半信半疑，看着那堆可爱的小狗，没有立刻离开。我抱起一只，这只小狗背部和头顶部长着黑色发亮的短毛，腹部、四肢和面部则是白色像绒一样的毛，偶尔有些黑色的斑点。它的耳朵耷拉着，乖巧地趴在盒子上，两只眼球黑黑的，有点害羞地望着，很招人怜爱，还有一个十分像Snoopy的鼻子。我心想，一定要买下它！好不容易，我说服妈妈，虽然价格比较昂贵，妈妈还是破例为我奢侈了一次。\ \

回到家里，我仔细看了它脖子上挂的牌子。原来它是一只纯种可卡犬，从英国来。我决定，长大后要买一只和它一模一样的狗。\ \

各种各样的狗给人不同的感觉。一般人养的狗，总是小巧可爱，让人很想把它抱在怀里，像小孩一样，它们会很听话，跟在你身旁，你跑它也跑；体型比较大的狗给人强烈的安全感，尤其那种毛色发亮又很整洁的大狗，主人带它出去特别有成就感，使人异常羡慕。\ \

我就是那种看到别人牵着狗，眼球就马上被吸引，心里又喜爱又羡慕，但只能干瞪眼自己不能拥有的人。\ \

走在繁忙的街上，经常会看到一些中年人摆个摊，放几只小狗卖。每遇到这些地摊，我都不得不停下摸摸狗，抱一抱，然后求妈妈买一只给我。每次我得到的回答总是否定的。妈妈总能说出一大串理由禁止我养狗。最让我没办法的是——“你去上学了，谁帮你看狗？”\ \

我又去问爷爷。谁知道爷爷不喜欢狗。奶奶在旁边附和说，有些人带着狗上街，看到熟人，就对狗说，叫“阿姨好！”，让人特别不高兴。这也令我很无奈。\ \

想起那篇课文《说不尽的狗》，中西文化差异的确很大，“狗”在很多情况下都使中国人感到羞耻。中国几千年深厚的文化底蕴啊！怎么给可爱的狗安排这样一个角色呢？真是令人不解！！！\ \

上高中后，我要很早起来上学。几乎每天早上，我都能看见住在附近的人出来遛狗，我多么希望有只小狗和我一起上学！记得有几天，早上比较凉，一个60多岁的很慈祥的老婆婆拉着两只不同种的小狗跑步，它们还穿了不一样的衣服，五颜六色，很有活力，我恨不得马上回头和它们一起跑步！\ \

但也有些狗不惹人喜欢。有些喜欢攻击人，有些不喜欢人，有些不理人。我觉得，一个地区的狗的生活状态可以反映出那个地区的人是否平和快乐。在乡下经常能见到孤独行走的狗、病狗或者伤残狗。在城市的某些角落，一些样子不招人喜欢的狗孤独地走着。\ \

去年我到学校报到那天，一下车就看到几只狗在天桥下徘徊。那时候我的心立刻凉了一截，我想那个地方肯定比较冷淡。又有一天我搭车回家。正在塞车中，路边突然出现一只瘸了一条腿的狗，人们都朝它投来异样的目光，而它好像不断躲避，不断地奋力地沿着路边跑，也不知道它要跑到哪里去，还是它要躲避什么。我在车上，看着，愣了，心都冷了。\ \

现在这个社会，是人影响狗，还是狗影响人？如果人爱狗，狗也会爱人。反之，人和狗都不会快乐。\ \

我好想快点长大，自己一个人住，住在喜欢养狗的人的社区，养一只属于自己的狗，和狗一起快乐地活着。 Created by X&Y
