欢乐谷之乐{初中时代的作文}

August 21 2006

欢乐谷之乐 \ 这个星期五，我们乘车来到深圳的欢乐谷游玩。\ \

进门后，我马上拿了一份地图，看里面有什么有趣的景点。只见地图上色彩缤纷，看得我眼花缭乱。总的来说，景区分八大主题区：西班牙广场、卡通城、冒险山、欢乐岛、金矿镇、香格里拉森林、飓风湾、阳光海岸，和玛雅水公园，园内共有100多个丰富多彩的游乐项目。\ \

我们小组里，只有李德明以前来过这里，所以就由他负责带路，给我们介绍一些有特色的游乐项目。\ \

计划好后，我们便直向冒险山里的太空梭飞奔。因为我们担心再晚一点，太空梭那儿就会有很多人排队。但没想到，当我们赶到时，那儿的队伍已经有几十米长了，我们只好边排队边和坐在太空梭上的人一起体验“飞天”的刺激。等了好久，终于轮到我们了！我既紧张，又兴奋，死死地抓住把手。倒数开始，“五，四，三……”我的心跳逐渐加速，坐在我旁边的同学也在大喊。’We’re the world!’我的话音刚落，太空梭“嗖”的一声就“起飞”了。我全身僵硬，摒住呼吸，绿树、高楼在我眼前疾速下滑。当达到最高点时，欢乐谷、深圳，大片景色一览无余，在这一刻，我感到无比的开阔，心旷神怡。但是，舒服没多久，太空梭又把我往下拉，旁边的同学继续尖叫，可强大的离心力却使我喊不出来。太空梭上下两三个来回后，终于停止了。我们回到地面，没有一个不说“真过瘾！”。\ \ 接着我们穿过情人园，尝试了森林攀爬，冒险山之旅就结束了。\ \

几分钟后，我们踏进金矿镇的金矿听音室。这里利用高科技全息音响技术，演示一个《古堡幽灵》的故事。暴风雨、开门、脚步、说话的声音，犹如就在我身边，使我感叹高科技音响的无穷魅力！\ \

感受过震撼的音响效果，我们又卷进了飓风湾的完美风暴。这是一个热门游戏，但当我排队看别人360℃旋转时，觉得很恐怖，最后还是坐在栏杆外欣赏同学们的表情。现在想起来，有点遗憾。\ \

最后一站，我们游览的是香格里拉森林。救援驼峰线、迷旋、蓝月山谷等等，都非常有趣，特别是斜屋，更让我感到人体的奥秘。\ \ 快乐、笑声充斥着这一天，欢乐谷的确带给我们不少知识和乐趣！ Created by X&Y
