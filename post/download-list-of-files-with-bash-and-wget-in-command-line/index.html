Download List of Files With Bash and Wget in Command Line

  • Why

Need to download a list of lecture slides from course page.

  • Example

for i in {11..19}; do wget -r -l1 http://someurl/slides_lecture$i.pdf;done;

  • Note

Inspect the url structure first. Easier to write one line of code than to create a general script.
