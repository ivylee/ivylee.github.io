Overpowering

感覺又好久沒寫奇怪的想法了。這次來點，有點不好的，類gossip

今天中午吃飯又是女生gossip的黃金時間，其中一個女生說，她宿舍最近11點鐘就關燈睡覺。我們的反應當然是詫异。她描述了同宿舍的某個女生的行為：11點到了，roommate1快上床睡覺吧~  訴苦的女生噴了一句：忍受不了這麽女王的人！自己想做什麽還要控制別人做什麽！此刻我的反應，與其他女生統一，默默的。其實我在想的是，終于有人發現了！終于有人對她不滿了！（是的很邪惡

自從我發覺不妥以來，我思考了很久應該怎樣描述這樣的特質，只有一個詞比較符合：overpowering.

A和B原來是初中同班同學，高一的時候，B和我是同班同學，A在另外一個班。 分班之後，A和我同班。大學時，A和我同班。

高一：我聽說A這個人，也聽B說過A很可怕很變態。初中的時候，如果有人考試成績比A好，A就會掐著那個人脖子說：為什麽為什麽….另外，A長得很像猩猩。

分班後：我和A不熟。group C 中午經常和A吃飯，後來有一天，group C 說不想和A吃飯了。具體的原因不清楚，或許是A太虛弱，太粘人。

大學：A突然好像跟我很熟，一天到晚拉著我去吃飯去上課。我開始不抗拒，可後來越來越不舒服。因為A總是用手挽著我的手，A的腳步從來不會考慮旁人，我總是被A拽著走。當然我從來不會拽A或者試圖抵消這種力量。

讓我對A不滿的，是A對人的態度。A自認為是公主/女王，所有其他人必須服從A，除了我。A習慣性地當面指著其他人做事，自以為是leader，掌權，in control，有責任。我認為這種是最crappy的leader. 我當然什麽都不說。A不敢指我做事，我不知道為什麽。其他同學曾經說過，可能是因為我在A跟前，A會感到我的強勢，how ironic! 我當然不會改變這種情況，因為對我有利。可有極少時候，我也隱約表示對A不滿，我想A也知道，我不在乎。

最後的結果是，我和A漸漸疏遠，很自然的，因為我要復習准備考試。A勾搭上了同班男生D，又是一段“佳話”。

曾經有一次和B小聚，說起A這個人，我和B超級有共鳴，我們都看到了A的問題，很明顯很突出很難接受。

漸漸我和A的共同朋友也覺得A不是個容易說話的人。A總是要別人去遷就她，別人必須按照她的意志做事，她達到此目的的方法，一是運用自己的power，指揮別人，強迫別人；二是向男生撒嬌，說好聽的話。兩者我都無法忍受。

A與同班男生D的關系，一直是我研究的對象。 D是個肌肉男，一直比較溫和甚至有點腼腆。不知道A和D是如何勾搭上的，因為事情發生的暑假我正埋頭別的事。A和D會在上課的時候一起坐，一起去倒水，有時候能看到他們在kiss，這讓班裏很多女生頂不順。在飯堂一起吃飯，排隊的時候也kiss，更加頂不順。飯堂這種充滿食物/异物味道的地方也有興致調情真是佩服。在AD關系裏，A必然是主導，強勢。A的弱點在于情感。A脆弱的時候，是A生病或者為前途/感情煩惱的時候。有一次A即將做手術，把我叫出走廊，寒風中我只穿了薄薄的睡衣，我多次說想進宿舍，A說不行！A的情感其實是很脆弱很幼稚的，D的作用應該在于此。在非情感方面A必然是主導。而D竟然是個如此軟的男生。說過了，都是周瑜黃蓋…

據我所知AD都是初戀，調低預期，觀望走勢。

在飯堂gossip之前，A有一次，感覺是好不容易找到我，然後趕緊跟我說，宿舍某個女生很晚睡，讓其他人都很不爽。我當然還是沒說什麽，隨便問一下開脫一下佯裝想想解決辦法。到了今天我確認矛盾比較大了。

坦白說有人看到A的特質我是很欣慰的。

gossip繼續，我問那個女生她們都是幾點睡覺幾點起來？回答說11點睡9點起。我覺得這未必太誇張了。再問A有沒有真的早起背單詞？沒有正面回應。那個女生說了一句：不想知道她的事。

要認識到A的特質并不難。當A和你有共同利益的時候，你很少會有感覺。當A和你有分歧的時候，感覺特別明顯。然後回頭再看，其實A的做事方法非常難以接受。或許有人會覺得A很有行動力，因為A有強烈的目標感，能夠達到目的。但只要考慮一下方法過程，不難發現A的手法讓大多數人都感覺不好。我不能認同A的做法。道不同不相為謀。

現在要觀望的，一是A宿舍的情況， 二是AD的關系。如果AD在畢業前分開了，必然很精彩。

黑暗的我。
