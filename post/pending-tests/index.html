Pending Tests

I wanted very much to try moblin and joolix on virtualbox. However, numerous problems came up…

Moblin, as everybody knows is suitable for netbooks. Well I haven’t got one but I’m trying out different os in case I’ll be tempted to buy one. First I downloaded the moblin .img, then google, set PAE in vbox, mount the .img and boot. Viola, scrambled screen. Today I updated vbox to 3.0.8, did the same thing and viola, installed. Reboot, stuck at the image

As for joolix, it’s an os developed by xiooli, based on archlinux. Problem popped up when I tried to generate the .iso image, said missing the mkisofs command. Google: pacman -S cdrkit. Iso is done, mount and boot. Viola, kernel panic! I’ve never encountered this before, had to give up.
