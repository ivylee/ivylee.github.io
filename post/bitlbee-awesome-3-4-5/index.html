折腾了bitlbee和awesome 3.4.5

突如其来的想法要用回irssi，于是就screen+irssi，进了熟悉的channel

发现了新东西：

  • awesome 3.4.5 is out.
  • bitlbee added support for twitter and facebook chat.

先说 bitlbee 。这个irc里的pidgin，之前介绍过，支持多协议(jabber, msn, yahoo..)，具体请见官网。想看看bitlbee的twitter是怎样的，众所周知的原因，我第一反应是弄个代理，后来发现自己是笨蛋。

跑到#bitlbee里问，原来直接连接国外服务器就好了，连bitlbee这个包都可以删了。

所以，最直接的做法是，在irssi(任何IRC client)连接bitlbee国外的 server ，推荐testing.bitlbee.org，SSL，支持twitter和facebook chat，然后就是register->identify->account add…

twitter:

account add Twitter <id> <password>

支持oauth，account add之后会提供链接到oauth页面，剩下的就不用说了。目前twitter的功能还很残缺，基本上只能看推发推，不过要有点希望啊~现在提供了3个mode, one/many/chat，不妨一试。

facebook chat

account add jabber <facebook_user_name>@chat.facebook.com

关于facebook chat，有个很恼人的问题，bitlbee的developer wilmer说，facebook不允许未登录过该facebook帐户的ip使用facebook chat，也就是表现在facebook chat account stuck at ‘﻿﻿﻿Authenticated, requesting buddy list’ =.=
