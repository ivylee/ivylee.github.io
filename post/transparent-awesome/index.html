Transparent Awesome!

Yep! today i’ve tweaked awesome a little bit and now it’s working with xcompmgr, producing nearly perfect transparent effect!

i’m on Arch Linux, if you’re lucky as i am, please refer to Arch Wiki on awesome 3, plenty of great stuff there. Oh I LOVE ARCH WIKI, AND KISS. always making things easy for newbies like me.

To enable transparency in Awesome, basically

  1. you get the git version of xcompmgr
  2. set it to start with x (in .xinitrc add “exect xcompmgr &”)
  3. using signals to control clients, write them in rc.lua.
  4. Voila! you’re done
  5. note: currently transparency using xcompmgr only works with clients. to make your wibox transparent, simply set theme.bg_normal = “#00000000” in your theme.lua. hey, it’s pseudo, but works fine for me.
