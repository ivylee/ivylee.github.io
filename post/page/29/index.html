欢乐谷之乐{初中时代的作文}

欢乐谷之乐 这个星期五，我们乘车来到深圳的欢乐谷游玩。 进门后，我马上拿了一份地图，看里面有什么有趣的景点。只见地图上色彩缤纷，看得我眼花缭乱。总的来说，景区分八大主题区：西班牙广场、卡通城、冒险山、欢乐岛、金矿镇、香格里拉森林、飓风湾、阳光海岸，和玛雅水公园，园内共有100多个丰富多彩的游乐项目。 我们小组里，只有李德明以前来过这里，所以就由他负责带路，给我们介绍一些有特色的游乐项目。 计划好后，我们便直向冒险山里的太空梭飞奔。因为我们担心再晚一点，太空梭那儿就会有很多人排队。但没想到，当我们赶到时，那儿的队伍已经有几十米长了，我们只好边排队边和坐在太空梭上的人一起体验“飞天”的刺激。等了好久，终于轮到我们了！我既紧张，又兴奋，死死地抓住把手。倒数开始，“五，四，三……”我的心跳逐渐加速，坐在我旁边的同学也在大喊。’We’re the world!’我的话音刚落，太空梭“嗖”的一声就“起飞”了。我全身僵硬，摒住呼吸，绿树、高楼在我眼前疾速下滑。当达到最高点时，欢乐谷、深圳，大片景色一览无余，在这一刻，我感到无比的开阔，心旷神怡。但是，舒服没多久，太空梭又把我往下拉，旁边的同学继续尖叫，可强大的离心力却使我喊不出来。太空梭上下两三个来回后，终于停止了。我们回到地面，没有一个不说“真过瘾！”。 接着我们穿过情人园，尝试了森林攀爬，冒险山之旅就结束了。 几分钟后，我们踏进金矿镇的金矿听音室。这里利用高科技全息音响技术，演示一个《古堡幽灵》的故事。暴风雨、开门、脚步、说话的声音，犹如就在我身边，使我感叹高科技音响的无穷魅力！ 感受过震撼的音响效果，我们又卷进了飓风湾的完美风暴。这是一个热门游戏，但当我排队看别人360℃旋转时，觉得很恐怖，最后还是坐在栏杆外欣赏同学们的表情。现在想起来，有点遗憾。 最后一站，我们游览的是香格里拉森林。救援驼峰线、迷旋、蓝月山谷等等，都非常有趣，特别是斜屋，更让我感到人体的奥秘。 快乐、笑声充斥着这一天，欢乐谷的确带给我们不少知识和乐趣！ Created by X&Y

狗{2005年寒假作文}

狗 2005年圣诞节，我和妈妈去逛中华广场。我们漫无目的地上了三楼，走着走着，突然我看见一堆好漂亮的小狗公仔。我拉着妈妈过去，妈妈也惊讶了。好漂亮！好逼真！好可爱！一问售货员，才知道这些是进口的，只在圣诞节时卖。妈妈半信半疑，看着那堆可爱的小狗，没有立刻离开。我抱起一只，这只小狗背部和头顶部长着黑色发亮的短毛，腹部、四肢和面部则是白色像绒一样的毛，偶尔有些黑色的斑点。它的耳朵耷拉着，乖巧地趴在盒子上，两只眼球黑黑的，有点害羞地望着，很招人怜爱，还有一个十分像Snoopy的鼻子。我心想，一定要买下它！好不容易，我说服妈妈，虽然价格比较昂贵，妈妈还是破例为我奢侈了一次。 回到家里，我仔细看了它脖子上挂的牌子。原来它是一只纯种可卡犬，从英国来。我决定，长大后要买一只和它一模一样的狗。 各种各样的狗给人不同的感觉。一般人养的狗，总是小巧可爱，让人很想把它抱在怀里，像小孩一样，它们会很听话，跟在你身旁，你跑它也跑；体型比较大的狗给人强烈的安全感，尤其那种毛色发亮又很整洁的大狗，主人带它出去特别有成就感，使人异常羡慕。 我就是那种看到别人牵着狗，眼球就马上被吸引，心里又喜爱又羡慕，但只能干瞪眼自己不能拥有的人。 走在繁忙的街上，经常会看到一些中年人摆个摊，放几只小狗卖。每遇到这些地摊，我都不得不停下摸摸狗，抱一抱，然后求妈妈买一只给我。每次我得到的回答总是否定的。妈妈总能说出一大串理由禁止我养狗。最让我没办法的是——“你去上学了，谁帮你看狗？” 我又去问爷爷。谁知道爷爷不喜欢狗。奶奶在旁边附和说，有些人带着狗上街，看到熟人，就对狗说，叫“阿姨好！”，让人特别不高兴。这也令我很无奈。 想起那篇课文《说不尽的狗》，中西文化差异的确很大，“狗”在很多情况下都使中国人感到羞耻。中国几千年深厚的文化底蕴啊！怎么给可爱的狗安排这样一个角色呢？真是令人不解！！！ 上高中后，我要很早起来上学。几乎每天早上，我都能看见住在附近的人出来遛狗，我多么希望有只小狗和我一起上学！记得有几天，早上比较凉，一个60多岁的很慈祥的老婆婆拉着两只不同种的小狗跑步，它们还穿了不一样的衣服，五颜六色，很有活力，我恨不得马上回头和它们一起跑步！ 但也有些狗不惹人喜欢。有些喜欢攻击人，有些不喜欢人，有些不理人。我觉得，一个地区的狗的生活状态可以反映出那个地区的人是否平和快乐。在乡下经常能见到孤独行走的狗、病狗或者伤残狗。在城市的某些角落，一些样子不招人喜欢的狗孤独地走着。 去年我到学校报到那天，一下车就看到几只狗在天桥下徘徊。那时候我的心立刻凉了一截，我想那个地方肯定比较冷淡。又有一天我搭车回家。正在塞车中，路边突然出现一只瘸了一条腿的狗，人们都朝它投来异样的目光，而它好像不断躲避，不断地奋力地沿着路边跑，也不知道它要跑到哪里去，还是它要躲避什么。我在车上，看着，愣了，心都冷了。 现在这个社会，是人影响狗，还是狗影响人？如果人爱狗，狗也会爱人。反之，人和狗都不会快乐。 我好想快点长大，自己一个人住，住在喜欢养狗的人的社区，养一只属于自己的狗，和狗一起快乐地活着。 Created by X&Y

自然的滋味{初中时代的作文}

自然的滋味 当你看着清晨晶莹剔透的露珠悄悄地在绿叶上滚动，在花瓣上嬉戏，你会觉得它是甜的吗？ 当你看到荒地上最后一棵树木苟延残喘，似乎在向你求救时，你能尝到酸的味道吗？ 当你看见树上的蝉正在努力地挣脱皮壳时，你会感到苦吗？ 它们就是自然的滋味。更多的时候，自然的滋味是很多种味道混起来的：小草的嫩甜和土壤的清香；眼看就要枯死的树木又长出了一片新叶；蓝蓝的天空如淡淡的水，朵朵白云像甜甜的棉花糖，搭配起来既和谐又有新意。 或许你会说是人的思想赋予了自然的滋味，但准确点来说，自然是人生的缩影，我们花一生时间所感受的酸、甜、苦、辣，都在大自然中表露无遗，这就是大自然的伟大之处。 因此，当我们吃糖时，可以和大自然一同分享；当我们吃到莲子心时，不但可从大自然中找到共鸣，大自然也会给我们一丝甜意，我们总能从大自然中找到我们想要的东西。 然而，当“三废”排出，氟利昂侵蚀大气层，各种各样的污物硬是搅进大自然里，大自然变成什么味道了？污浊的东西入侵大自然，如同污浊的东西入侵人类，不仅是躯体，而且是灵魂。可谓“百感交集”？ 一切不合谐的音符都应该删除，一切不合谐的杂味都应该剔除。我们要的是一个清洁、纯正、有滋有味的大自然，而不是一个麻痹味觉、不知所谓的什菜锅。因此，请厨师们不要再相信所谓吃了能生钱的偏方，让大自然自己调味吧，这才是最和谐，最美味，我们最想要的。 Created by X&Y

表弟{2006暑假随笔}

表弟 我的表弟11岁，要上六年级了。他妹妹比他小两岁，比他高一个拳头。 表弟不爱吃饭，不用吃饭他最高兴。他最喜欢的是车，可以一天到晚坐在电脑前玩极品飞车，看电视剧或在外面走时他一看到车就能马上讲出型号，可我们家没什么人懂车，自然也没什么人回应他。 表弟很瘦，皮包骨，但看上去挺聪明，听说成绩也很好，也挺听话，和其他小孩比起来，不显得任性。在深圳住时，表弟依照大舅的吩咐，每天从一本叫《弟子规》的书上找一句话打在Word文档里，然后E-mail给他。表弟的确是每天都打，换了我，干脆一次过打完算了。他爸爸，也就是我大舅，做事非常细心谨慎，这本《弟子规》讲的是晚辈应该如何对待长辈，大舅的意思应该就是让表弟天天都学习。 我和表弟的共同兴趣除了电视剧就是极品飞车。说真的，我不怎么了解车，我只是喜欢速度的感觉。表弟知道的比我多多了，哪辆车好，怎么装配件，在哪里刹车，哪里有捷径……简直是游戏专家。在我玩游戏时，表弟在旁边看，突然说：“这里走近路！”“刹车！”“后退！”“啊！要死了！要死了！我不看了！”表弟的叫声吓得我异常紧张，无法集中精神。 我在厅里睡的一个晚上，天气不好，有台风，下大雨。半夜，我朦胧之中听到“嗡嗡”的声音，还听到2次，我心想肯定是蟑螂！于是把头蒙得严严的，直到第二天醒来。 一天早上，我们吃早餐时，表弟突然叫道：“蟑螂！”我顺着看去，是只不小不大的蟑螂，我马上缩脚上沙发，外婆急忙问：“哪里？”表弟指着，那只蟑螂马上跑。外婆手拿拖鞋“追杀”蟑螂，几次攻击后，蟑螂多进了床底，外婆无功而返。我问表弟：“你也怕蟑螂啊？”他说：”不是啊。打他把拖鞋底搞得很脏。”真无奈！ 表弟十分害羞。那天晚上家里没酱油了，我和表弟到楼下去买。在电梯里，他问我：“等一下，你说还是我说。”我猜到他这样问，肯定是不想说。于是我说了。我们和外婆出去时，外婆说：“他说我的普通话不准，不要我讲，他自己又不讲。真是的！” 别人问表弟一些事，他总是支支吾吾。我问他：“你有没有去过游泳？”他说：“没有。有吧。我不知道。” 其实表弟就像我大舅，不会主动和别人说话，说起话来又小心翼翼。但他 还是很可爱的，活泼而不顽皮。 我好期待他长大，希望他是个高高瘦瘦的英俊小子，内敛其实是一种魅力。 Created by X&Y

读《围城》

读《围城》 我读《围城》有一气呵成的感觉，但越读越不明白，到最后更是郁闷。 或许那个时代人们的生活态度和现在有很大不同，像是现代思想和古老的封建思想混杂在一起，使得人们做事情很奇怪，我很难理解。我认为《围城》讲的不仅是婚姻，还有更多的内容。 读完小说回头想想，其实主人公方鸿渐像是在虚度年华，一事无成。这个人留洋弄个假文凭，回国找不到工作，失恋后跟着朋友游荡，草率地和一个没什么感情的女子结了婚，婚后吵吵嚷嚷，最后一拍两散，他又心存希望去投靠朋友……似乎是个恶性循环。赵辛楣说得好，方鸿渐这个人不讨厌，可是全无用处。他有点阿Q精神，生活潦倒也就凑合着过，实在不行了安慰一下自己然后就走；他自己没本事，又不愿接受别人的帮助，硬是有骨气，有傲气，不肯踏踏实实地干，最后爱情、事业、钱财什么也没得到。 从《围城》中生动巧妙的描写和偶尔几句英文法文的运用，可以看出作者钱钟书是个见过世面而且非常有趣的人。他想象力丰富，能用简练的语言把人物的外貌、心理活动和环境描写得十分恰当而生动，使读者身临其境，身如其人。有很多处作者的语言幽默尖锐，讽刺意味十足，十分值得摘抄下来，如： 描写沈太太：她眼睛下两个黑袋，像圆壳行军热水瓶，像是储蓄着多情的眼泪，嘴唇涂得浓胭脂给唾沫带进了嘴，把黯黄崎岖的牙齿染道红痕，血淋淋的像侦探小说里谋杀案的线索，说话常有“Tiens ! ”“O la, la！”那些法文慨叹，把自己身驱扭摆出媚态柔姿…… 作者语言的精妙实在是令人佩服！ 小说的最后一句真正引起读者的深思，“这个时间落伍的计时机无意中包含对人生的讽刺和感伤，深于一切语言、一切啼笑。”我看了不是很明白，想是因为年龄的关系，或许大了就懂了。 Created by X&Y

读《边城》

读《边城》 《边城》的结尾，我觉得很难理解。为什么两个原本相爱的人就这样不了了之了？在那个如此理想化的淳朴乡村，一切生活本应如意，难道真的是“凡美丽的都不容易长存”？那么历史悠久的小城里，人们又怎样创造和谐美好的生活？ 作者对乡村小城生活情态的描写，给我留下很深的印象。简单易懂的文字串联起来，使人津津有味地一直读下去，脑海里更浮现一片繁忙安详的乡村生活景象。一切都是那么和谐美好，依山傍水的环境，原始古朴的情调，淡雅醇厚的气息，环境和人互相融合，这样的生活实在是令人羡慕。 主人公翠翠和祖父相依为命，体现了一种相互关怀和体贴的亲情美。翠翠自幼丧失双亲，但在爷爷的悉心抚养下，她灵性，乖巧，善良，纯真，有一种质朴、淳厚，一种纯自然的美质，她就像是大自然的灵魂。祖父是一位老船夫，50年来尽忠职守，他重义轻利、重友情远钱财，宽容善良，默默地为人们奉献，体现了当地人们淳厚的品德。 对于顺顺和他两个儿子的描写，更加体现了边城人的质朴、淳厚的人性美。顺顺慷慨大度，侠情满怀，重利轻义，教子有方；大老天保和二老 送不骄惰，不浮华，不倚势凌人，正直勇敢，都能配得上纯美的翠翠。 于是，矛盾产生了。大老向翠翠祖父提亲，祖父又不能决定，翠翠爱上二老，也因害羞而没有明说，二老决定和大老公平竞争，可谁知此时大老溺水而死。事情变得越来越复杂。二老的态度变了，顺顺也不想提婚事了，祖父感到为难，翠翠陷入痛苦之中，最后祖父去世，翠翠生命中最重要的东西失去了，留给她的是无尽的悲哀与伤痛，还有那不知何时才归来的心上人。 对于作者写的这一段，我很难理解。在乡村，女孩15岁便可出嫁，祖父也默许了，翠翠怎么就不承认自己与二老相爱呢？翠翠不表态，使得祖父不好说话，大老也等得心急，后来二老的加入不可避免地打击了大老的信心，这些都与大老后来的死有一定关系。另外一个很重要的问题，为什么大老死后顺顺和二老对祖父的态度突然冷淡？乡村人的迷信思想是主要原因吗？我认为大老的死和二老有一定关系，要追究起来，许多人都有责任。为什么二老就不能放下自己的旧思想去找翠翠呢？他仍然爱着翠翠，可就不肯娶她，还不负责任地走了，这是不是反映了乡村人的愚昧无知？还是太重情义？这未免有点过分了吧？ 爱情怎么会因为一个人的死而灭亡？这是最令我不解的。 “凡美丽的都不容易长存”这句话没错，但我认为，在《边城》里，作者把美丽的爱情换成痛苦而长存了，这与他所描写的幸福和谐的边城形成十分明显的矛盾！但也许可以说，生活本来就是矛盾的统一体。 书上说：“《边城》是一部充满理想主义的小说，它所描写的故事是现实的，更是理想的，是现实与‘梦’的结合。”也许结尾理想的破灭，更使人感到理想的珍贵。 Created by X&Y

谦逊{初中时代的作文}

谦逊 翻开今年的最后一期《读者》，卷首语是《谦逊》。 “人们称谦逊为一切美德的皇冠，因为它将自觉的纪律、天职、义务以及意志的自由和谐地融会到一起。一个谦逊的人如果将自己身上一切值得赞扬的东西都看作是应该的、理所当然的，那么他就会将纪律当作真正的自由，并且为之努力奋斗。” 苏霍姆林斯基的这篇文章令我感触良多，他谈到的问题似乎就是我和我们班的写照——自高自大，把对于某种复杂事物的模糊的、肤浅的、表面的印象当作知识，严以待人，宽以律己……每一句精辟的分析，都使我看到了谦逊的高尚，自大的无知和愚蠢。 这又使我想起了星期四那天，我们班在篮球赛中输给七班的情形。开始我们班并没有派主力队员上场，比分一直相持在十几分。紧接着，我方数次的走步犯规，都打断了进攻的高潮及连贯，而七班却很少犯规，导致七班领先我方三、四分。比赛越来越激烈，我方犯规次数越来越多，投篮命中率也不断下降，队员们更加着急，但失误也更多了。随着铃声终场，我们班输了。 虽然裁判是七班的体育老师，而且不断鸣哨指我们班犯规，很有嫌疑，但是总结起来，这次我们班的失利很大的原因是我们轻敌。自初一以来，我们班的篮球赛频频报捷，这次面对七班，当然无所畏惧，甚至在上半场不派主力迎战，队员们的确十分自信。可到了下半场，眼看比分逐渐落后，主力才开始上阵，可惜为时已晚，队员们浮躁不安，乱了阵脚，反而七班的队员稳扎稳打，顺利进攻。 虽然战果不尽人意，但我从中却获益良多。我们应该分清自信与自负，正确看待自己的优缺点和估计自己的能力，经常自我反省、自我教育，虚心学习他人优点长处，宽以待人，严以律己，这样才能够进步，才能够做一个谦逊的人。 Created by X&Y
