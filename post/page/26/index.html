风格与变化

今天上网，发现周杰伦的新专辑已经在网上可以下载了。我第一时间决定去尝尝鲜！毕竟是周杰伦啊，还是十分令人期待的。但同时，我又不敢抱太大希望。 想起上一张专辑《十一月的肖邦》，给我的惊喜实在不多，反而让我觉得周杰伦已经跳不出自己的框框了。 但无论如何，这张《依然范特西》我听定了！ 专辑取名《依然范特西》是否含有延续以前专辑风格的含义呢？不知道追求曲风变化的人对这张专辑是否满意呢？ 由于这张专辑是热中之热，我很快就下载完了。按下Play按钮，我开始仔细地欣赏。 想不到专辑中最令人意外的不是周杰伦，而是一个叫费玉清的人。在《千里之外》这首歌中，他的声音与旋律完美地融合，很好地表现了歌曲的意境，而周杰伦则更像在陪衬。但回头想来，周杰伦能够请到费玉清来与他合唱，实在不是一件容易的事，而且可以共同演绎一首相当优秀的作品，周杰伦也是功不可没。 另外一首挺有趣的作品是《迷迭香》。不知道算不算是爵士曲风，但很明显脱离了周杰伦一贯的R&B和Rap的调子。虽然周杰伦的声音似乎和音乐没能擦出什么火花，比较平淡，但这也是一大尝试。 其他歌曲与他以前的感觉比较相近，在细微的地方有些改进。我还发现周杰伦在某几首歌中咬字清晰了，让我听起来没那么别扭。 总的来说，这张专辑中的歌曲给我留下的印象还是不错的。我认为这张专辑里周杰伦所尝试的变化比他以前每一张专辑都多，至于是好是坏，那就见仁见智了。 风格与变化，是一对相对矛盾而相对稳定的词语。每个歌手都希望建立自己的风格，但一旦风格成熟稳定后，对于变化的需求又与日俱增。有的乐迷就喜欢一种感觉，而有的确喜爱新鲜感。变化中的稳定与稳定中的变化该如何衡量？这实在，就像我刚才说的，要见仁见智了。 我一直希望周杰伦会在音乐上有所突破，而又希望F.I.R.保持自己积极向上活泼的旋律，我能沉思在Chet Baker的忧郁吟唱中，却又充满希望地在陈奕迅的音乐中寻找刺激…… 歌手要满足乐迷的愿望真是很难。100个人看哈姆雷特，就有100种感受。音乐中没有永恒的静，只有永远的动。在无数的风格和无数的变化面前，我们可以尽管尝试，最重要的是找到自己喜欢的感觉。 Created by X&Y

教师节 Teachers' Day

Sept.7th,Friday. 今天，到处都在“庆祝”教师节。黑板报，讲台，都为了教师节而特地布置，还有仪式上的问好。为什么我们几乎天天都见到老师，偏偏只是这一天才“祝老师教师节快乐”？ 但是，教师节有2个好处：第一，可以见到以前的老师；第二，可以见到以前的同学。 回到十六中，一切都是那么熟悉，一切又物是人非。眼前看到的是1年前看到的景象，感觉却十分不自然。即使见到认识的老师，也总会有种被遗忘的感觉，我甚至不知道如何开口跟他们说话。 反而，见到许多旧同学，像是把一整年的郁闷都释放出来。我们交谈的方式和谈论的话题如有默契般地一致，我就想回到了以前，随心所欲，畅所欲言的感觉，丝毫不用担心身边的人听了会有什么样的想法，也不用担心无意中伤害了某些人。 其实我只和一个老师谈了话，而且她很自信地叫错我的名字。我不敢去见另一位老师，大概是因为愧疚而胆怯。 我和同学在M记待到快9点。同学间最珍贵的是那种熟悉放松的感觉，对我来说最宝贵的是他们没有变，变也只是变得更好了。或许，我变了，我感觉到自己变了，变得封闭，不愿多讲话，但当我回到他们中间，就像找回了3年前的我，幸好3年前的我没有改变。 我在去十六中的时候，突然想给以前的老师发短信。我在电话本里找了一遍，发现老师的电话并不多。我想了很久，最后只是发了一句很简单的问候语，给以前的数学老师。令走向十六中的我忐忑不安的心情有一丝愉快的，是数学老师回了一条短信：“谢谢”。 难道不应该每天都是教师节吗？ Created by X&Y

关于高二分班

8月31日，我终于知道分班了。我在7班，纯化学班。我看了7班的名单，只有4个我原来的同学，其他人都不认识，而且某些人在我想象中十分可怕。 我原来的班里有许多同学进了创新班，也就是重点班，都是选物理的。我想原来普通班里选化学的都没有进创新班，这看出华附对物理和化学的重视程度差很远。 今天我看到了我们班的所有同学，感觉一般，没有以前那么有趣。班主任我比较满意，是我原来的化学老师，应该算是我原来所有的老师中给我感觉最正常的一个。 书都发下来了，一共19本，惊人的数字！其他科的老师还不知道，但我想不会很好，也不会很差。希望数学老师不是那个原来教7班的。 我原来的数学老师竟然又跑到高一去了。31号一个英语老师还跟我们说我们原来的数学老师是5班的班主任，这个英语老师真是奇怪！好在我的英语老师和他不是同一个人。我想英语没有完全按照成绩来分班，而主要是按行政班，不过还好，听说我的英语老师不错。 Created by X&Y

男篮世锦赛——中国队输了

我又是从第二节开始看比赛。一开电视机，中国队25比15领先10分，吓我一大跳。我心想，这怎么可能？一定是尤纳斯的战术把希腊队克制住了！我正这样想着，中国队就连续进攻不力防守失利，希腊队一下子就把比分追上来。GRK突然间换了全场紧逼防守，以致中国队失误百出，后卫运球从底线带到前场都成了大问题。GRK队员防守非常坚决，中国队一节就有了10个TO，GRK的STL不下10次，而且每次STL得手后就是快攻上篮，以致分差一下子拉开。 眼看场上急剧下滑的形势，我着急地想，尤纳斯快点想个战术来破他们的全场紧逼啊！他不是很熟悉希腊队吗？一定有什么好办法能攻破他们！ 但是，场上中国队的表现令我大失所望，犯过的错误一犯再犯，毫无改变战术之意。他们总是不断地失球，即使有了组织进攻的机会，他们也是不能好好把握，仓促出手，进攻成功率极低。GRK在坚固防守的同时，几个优秀的投手也使他们的得分节节上升。第三节单节，中国队仅得6分，GRK得了26分！ 第四节尽是“垃圾时间”，中国队一直在重蹈覆辙，到比赛后半段GRK也不严密防守了，最终中国队以31分败于欧洲冠军GRK。 这场落败，我相信不是尤纳斯的错，尤纳斯为中国赢得了一个近乎完美的开局，而中国队队员没有找到能够打破对手紧逼防守的有效方法，致使他们不断地丢球和丢分，这大概和他们的身体素质有关系。在与SLO的比赛，因为那是出线的生死战，所有中国队员都奋力拼搏，赢了比赛，输了体力，但这也是不得已。对GRK的比赛一样是淘汰赛，中国队开局很好，但对手一旦紧逼，一旦气势压倒了中国队，我感觉中国队连拼的信心和力气都没了，身心都输给了对手，也算是“心服口服”。 如果中国队在小组赛中能先胜一场，稳稳当当地确定出线，再好好地调整准备和对手的比赛，那么中国还是有可能进八强的。从这次世锦赛看来，中国队还是经验不足，在各个方面都与已进八强的队伍有明显的差距，但是我相信，在尤纳斯的指导和带领下，中国男篮会不断进步，有一天真正成为中国人们的骄傲！ Created by X&Y

太阳系九大行星→太阳系八大行星

前几天我在新闻上看到IAU大会召开了，满心希望太阳系行星家族会增加新成员，但没想到结果和我的预想相差甚远。 讲到太阳系行星家族的新成员，我最熟悉的是2003-UB313。高一上学期，地理老师给我们的第一个大任务就是围绕一个课题研究作一次课件演示。我们组选到的是“太阳系的‘第10颗行星’”。我们选这个题目主要是因为它是一个很新的发现，能够引起人们的好奇心。 演示课件主要是我做的。我先上网查资料，渐渐发现有2个新发现的天体都有被称作“太阳系的第十大行星”，一个是Sedna，另一个是2003-UB313。两个天体都是由Mike Brown宣称它们有资格被列为行星之一。但是我经过比较，认为2003-UB313体积比冥王星大是有力的依据，它成为行星之一的机会比较大，因此就选择了它来作介绍。关于2003-UB313的详细介绍，可以看 IAU的决定令我感到意外，但是理由能够令我信服。IAU对行星的定义进行了补充，认为行星的轨道应该是确定的，不应受其他天体的影响。冥王星的运行轨道正是因为受海王星的影响而成椭圆，而且轨道面与其它八大行星有很大的偏角，Sedna和2003-UB313的轨道与八大行星相比更是偏得厉害。把冥王星否定后，行星家族的共同点同面性和近圆性就能对所有行星适用，增加了它们的整体统一性。 至于把冥王星、Sedna、2003-UB313等归为矮行星，我还没有作更深入的了解。 另外，要将大量的教科书、科普书籍、科学博物馆里的“太阳系九大行星”改为“太阳系八大行星”的确非常需要时间，但我相信，只要人们对行星有更加深入的认识，接受“八大行星”并不是问题。 从几十年前发现冥王星，把它归为太阳系行星之列，到现在通过投票将它除名，反映了人类对太阳系，对天体运动的认识的过程。至今行星还没有确切的定义，希望我们能够发现越来越多的天体，从各种细节认识宇宙，逐渐完善行星的定义。也许到那时候，许多谜团的答案就解开了。 Created by X&Y
