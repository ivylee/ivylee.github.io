世锦赛——中国（CHN）VS斯洛文尼亚（SLO）

真是没有想到，中国队真的赢了，奇迹般地赢了！ 我从比赛第二节开始观看。第二节中国队防守不到位，SLO队员勤于绕圈返跑，而我们几个防守球员都有相同的弊病——只会跟着别人跑，总站不到对手面前防对手的中投或三分远投；另外防守篮板大多数被对手毫无压力紧紧地拿住，以致经常有连续的进攻机会。防守不好，进攻的机会就少，在中国队球员中，姚明依靠个人能力打出几个漂亮的2+1，杜锋也有2个及时的三分球，但其他的进攻就显得松散。第二节中国队落后一度达到十几分，幸好结束前又轻微缩小了。 第三节中国队打得非常出色，在防守上有了明显的改进。后卫都加紧脚步，把SLO的队员跟得无法出手，这也是中国队教练尤纳斯的战术——外线紧防，逼他们打内线。虽然SLO几个中锋身体素质都很好，还有一个效力于马刺队的“内…”，但姚明拼尽全力，努力地防守和得分，这是十分值得中国队其他队员学习的。中国队防守严谨，进攻也比较细心，全队积极得分的态度使中国与SLO的分差不断缩小，更曾有3分的领先优势。但第三节末SLO凭几次干净利落的进攻，又以微弱的领先优势结束。 第四节中国队开始几分钟打得不是很好，防守上似乎有所放松，进攻也连续几次不奏效，其中也有姚明体能严重下降的原因。数次暂停后，中国的防守有了改善，比分不断接近又不断拉开。王治郅的出场略显晚点，一上场就一记Run and shoot的三分，接着一次快攻上篮得分，然后和姚明配合几次防守成功，让中国队看到了胜利的希望。但是，SLO队员在关键时刻没有手软，也是几次干净利落的进攻，取得了4分的领先优势，场上剩下不到2分钟。艰难的时刻现在才开始。朱芳雨先投进一个3分，但刘玮一次防守犯规使得对方2罚1中，中国队还落后2分，场上只剩下5.8秒。只见刘玮从后场开球，王仕鹏拿球就跑，快到3分线，两个SLO大个围上来，出手！球划过高高的一条弧线，稳稳地进了！真是出人意料！SLO队员都傻眼了。王仕鹏被中国队队员紧紧地抱着！队员们都兴奋极了，尤纳斯也异常激动，挥舞着双拳。 向来不看好中国队的我不得不承认，今天中国队的表现实在非常出色，尤其是第三节。尤纳斯挥舞着双拳，似乎不仅在告诉自己中国队赢了，更像是在肯定自己的选择和眼光。直到今天我才明白，为什么尤纳斯对中国男篮抱有这么大的信心和期望，为什么他愿意放弃其他的好差事来担任中国队的主教练，是因为中国队有能力，有能力成为一支强队，只是没有得到很好的指导和训练。今天的中国队真正地给了球迷信心，真正地使人相信，中国篮球正在进步，他们有能力勇往直前，不畏惧任何困难！ 中国出线后的对手绝不好打，是欧洲冠军希腊队。面对他们华丽的打法，防守是重任，希望进攻效率能够有所提高，下下策是后卫冲进内线上篮，要么得分，要么造犯规。期待尤纳斯有好的战术，中国男篮全力拼搏，04年奥运赢了塞黑，06年世锦赛也可以赢希腊！ Created by X&Y

2004年7月18日{初中时代的暑假日记}

2004年7月18日 晴 今天下午，我刚睡醒，走到厅里，发现爷爷一个人站在鞋柜边摆弄着，不知道在干什么。 我走上前去，看见鞋柜上有一把大剪刀、一把大尺子、一支红水笔、一把锯刀（像是从什么地方拆下来的，没把手）、几片长方形的塑料片或是铝片，真猜不出爷爷在干什么。于是我问爷爷：“爷爷，这些是塑料片还是铝片？”爷爷回答说：“这些是铝片，看起来是有点像塑料。”“那您在干什么啊？”爷爷微微笑了笑，神秘地说：“呵呵，不告诉你，你肯定猜不着，这东西用处大着呢！”哈哈，爷爷还装神秘！我在他旁边仔细观察观察，我就不信我猜不着！ 铝片比较容易定型。只见爷爷量了量尺寸，在铝片长边的两端留了一段距离，用红笔点上两点，然后拿锯刀分别锯开垂直长边、以两个红点为端点的线；接着把分开的三部分往里折，就像做个长方形的盒子一样。我猜道：“爷爷，您在做个盒子吧？”爷爷仍是笑了笑，说：“保密。”“不是盒子是什么？这不是一个无盖的长方体吗？”“嘿嘿，看下去就知道了。” 我还是第一次看到爷爷这么神秘，这东西究竟是什么呢？有什么用呢？ 果然不是盒子！因为爷爷只把2条长和1条宽折起来，这样就装不了东西。那会是什么呢？是用来铲东西的吗？也不是，这个东西容积不大…… 正想着，爷爷用剪刀把两端折入的小长方形剪去两个小三角形，再把两边往里压一压。这时，爷爷拿起做好的东西，走到浴室里，在抽风机侧面的空隙比划着。 “哦，原来是这样！”“哈哈，想不到吧？退休了就该干这事！” 上星期浴室装修，抽风机侧面贴不了瓷砖，所以爷爷想用这个东西嵌进去，使浴室看起来美观些。爷爷真是有创意，难怪他说我肯定猜不着！ 第一次“试嵌”，这个东西的角大了些，嵌不进去。经过爷爷的“整形”，第二次它就紧紧地嵌了进去，大小、位置刚刚好。看来爷爷对自己的创意也挺自豪，满脸欣喜，一边用俄语说着“好极了！”，一边喊着奶奶来看他的杰作。 我以为把这东西嵌进去就大功告成了，谁知爷爷又拿来一小罐白漆和一支旧毛笔给它上色。 真正大功告成了，爷爷脸上露出了满意的笑容，再一次用俄语赞不绝口“好极了！”“好极了！”…… 我真佩服爷爷，对于这样一点小事，都想得如此周到、细致，换了我肯定一早就不耐烦了。爷爷不愧是爷爷！ Created by X&Y

2004年7月24日{初中时代的暑假日记}

2004年7月24日 雨 今天下雨，最好就是静静地坐在家里看书。 在一本书上看到这么几句话，觉得挺有意思： 学习要加，骄傲要减，机会要乘，懒惰要除。——现代人的人生运算 不会思考的人是白痴，不肯思考的人是懒汉，不敢思考的人是奴隶。——一句拉丁美洲谚语 我们可以躲开一头大象，却躲不开一只苍蝇。——使我们不快乐的，常是一些芝麻小事。 第一句简直就像在说我，虽然在学习上得到了不错的成绩，但是仍要加把劲，不能骄傲自满。我马上把它抄下了，的确十分有意思。 第二句话是关于思考的。但我看到第三小句“不敢思考的人是奴隶”时，我觉得很可悲。或者是拉丁美洲的人常被当作奴隶交易，才能如此简洁地道出奴隶的痛苦。不敢思考的人就像没有了灵魂，难道作为一个奴隶不仅要为主人干活而且还要献出自己的灵魂吗？这是十分残忍的，比杀了一个人还残酷。我真庆幸自己不是生于那个时代，我们要做一个善于思考的人，这样我们才能进步。 第三句是谈快乐的。的确，常是一些芝麻小事使我们不快乐，但是，也有些芝麻小事能使我们很快乐。比如说，春天到了，树木开始发新芽，这些周而复始的事情或许没什么人会注意，但我却觉得很高兴，因为新的生命又降临了。我时常抬起头仔细地看着树叶，那种翠绿和光泽，是颜料无法调出来的，很独特。其实，快不快乐，是看你从哪个角度去看事情，看问题，没必要搞得自己不开心，所以，快乐一点吧！ 写完这篇文章，我又快乐了一点。因为我思考了，写出了自己想写的东西。 Created by X&Y

2004年8月16日{初中时代的暑假日记}

2004年8月16日 晴 今天，我和妈妈去游览烈士陵园。 其实，我们常常去烈士陵园，可对它却没有详细的了解，因此今天的主要目的是游览全部景点，从整体上认识烈士陵园。 我们收拾好必需品，大约10点多，就来到了烈士陵园正门。我们买了地图后，就坐在树荫下画路线，尽量不重复地游览。 我们从正门出发，沿着苍松翠柏的阴凉，经过四烈士墓，来到了叶剑英元帅墓。叶剑英同志是中国人民解放军的缔造者、中华人民共和国的开国元勋之一，他为中国人民解放事业和社会主义建设事业奉献了毕生精力，建立了丰功伟绩，十分值得我们尊重。 上了台阶，我们走近了烈士陵园的最醒目的标志——广州起义纪念碑。只见一只粗壮有力的大手握着一杆长枪直冲云霄，寓意“枪杆子里面出政权”，正是广州起义中烈士们的浴血奋战，才能换来我们今天的幸福生活，因此碑身刻有邓小平同志题字“广州起义烈士永垂不朽”。 顺着台阶走上去，就看到了广州公社烈士墓。墓上长满青草，幕墙正面有朱德同志的题字“广州公社烈士之墓”，可以想象到当时死伤人数之多，许多甚至是无名的，都被埋在这里，值得我们永远尊敬。 下面就到了血祭轩辕亭。它是为纪念“刑场婚礼”中的周文雍、陈铁军视死如归的英雄气概而建的。 中苏人民血谊亭和中朝人民血谊亭是为纪念广州起义中英勇献身的朝鲜和苏联烈士而建的。亭内分别有两块石碑，由叶剑英同志和陈郁同志分别题词颂扬这些革命战士。 这样，烈士陵园内的景点就基本上游览完了。我们在三角亭休息了一会儿。除了游览景点，烈士陵园也是老人们的乐园，他们在这里跳舞、唱歌，十分开朗健康。休息完后，我们就走出东门回家了。今天的游览虽说没多大新意，但我学会了从不同角度看事物，我想以后游览公园都应该像今天这样，才能够学到知识。 Created by X&Y

2004年8月23日{初中时代的暑假日记}

2004年8月23日 晴 今天，原计划舅舅和他的同事从深圳来到广州的广东美术馆看国际著名设计师靳埭强的设计展，舅舅也约了我一起去。 我和妈妈吃完午饭后，就来到广东美术馆。我们先看了这个设计展的宣传海报，它的全称是“生活·心源——靳埭强设计与艺术展”；再看看时间，我们正好赶上了配合此展览的第二个讲座——靳埭强的设计与艺术，于是，舅舅把我们带到了报告厅。由于舅舅所在的公司正是靳埭强先生在深圳开设的分公司，因此我有幸坐在第一排，近距离地听大师演讲。 在我们入场后，陆续又有许多人进场，一部分人甚至站在场边或坐在地板上，可见靳埭强这个名字有多大吸引力。 讲座分几部分进行。靳先生讲述了他如何从一个小裁缝转变成一位设计师，以及中西文化对他创作的影响等等，同时他也是一位水墨画家，他常常将水墨融入到设计中，别树一帜。 最特别的就是一个红点。起初是他的水墨画老师用以代表荷花，后来靳先生赋予了红点多种含义，有“心”、“莲花”、“红日”、“伤口”等，富有新意。 虽然有些作品我不大明白，但我知道了靳先生认为设计最重要的一点——创意。无论何时，创意都是最重要的，有了好的创意，就成功了一半；而创意的来源就是我们身边，生活、社会、文化、大自然，只要细心观察，就会有收获。 靳先生不仅是一个设计师，而且是一个商人。在最后的问答过程中，靳先生回答设计师提出的客户不满设计成果的问题，主要以与客户沟通为主；从设计一开始就与客户商量，了解产品、做市场调查等等。许多设计师只讲艺术，认为自己设计得很好，不配合产品市场，而靳先生不然，这也是他能在商业上成功的原因之一。 讲座结束后，我竟有幸和靳先生共进晚餐。舅舅公司的所有成员和我坐车来到离美术馆不远的酒店吃饭，虽然有点紧张，但想起刚才一拥而上排队拿签名的听众，我已是十分幸运了。 后来，舅舅还帮我拿了一本有靳先生自己亲笔签名的设计作品集。 这个晚上的确很难忘。我竟然与大师如此近距离地接触，这是多少人梦寐以求的，而我却如此简单地与他吃晚饭，这大概就是缘分吧。靳先生说过，能加入他们的公司工作的人都是有缘分的，希望我日后也能有机会向他学习。 Created by X&Y

2004年8月3日{初中时代的暑假日记}

2004年8月3日 晴 昨天下午，我和妈妈去游泳。 出门前，妈妈叫我把厨房的门关上。“哦。”我走上前，“呼”一下子把门给拉上，扭头就走。可妈妈却发现门拉不动了。“你自己过来看看！”妈妈很生气地说。我心想，大事不妙，于是马上跑过去，使足了劲儿，左拉拉，右推推。门还是拉不动，我想大概是卡住了。妈妈可气极了，因为以前也有两三次发生同类型事件，都是爸爸或妈妈弄好的，这次爸爸去出差了，妈妈也无从下手，因此妈妈说既然是我弄坏的，我就要负责任。我自知理亏，只好想法子。 一出门我就在想这件事怎么办，游泳时间净在泡水，晚饭只能在外面吃。 晚上回到家，我和爷爷通了电话，爷爷爽快地答应了今天到我们家帮我们修门。 今天早上我还没睡醒，就听见爷爷按门铃，只好赶快起床，洗把脸清醒清醒。只见爷爷拿着个大袋子，里面装着两把钳子、十字螺丝刀……我惊讶道：“哇，带这么多东西！”“当然，怕你们家没有嘛！”爷爷想得可真周到。 我和妈妈把弄坏门的经过告诉了爷爷，爷爷想了想，“那现在就是门卡在上边的轨道。”“对，那接下来给怎么办呢？现在门动不了。”这是我最大的疑问。我认真分析了原因——门偏离了原轨道，从门是斜的而且拉不动可以看出；要重新装门就必须先把门拆下来，但是，怎么拆？有工具可用吗？ “试一试吧。”说着，爷爷握紧把手，扎好马步，使出全身的劲，“一震”，门脱离了地下的轨道。我和妈妈都惊叹道：“哇，爷爷好大力，真是厉害！”爷爷微微地笑了笑。 现在上面轨道已经对上了，可下边轨道顶端的木块高了一点，把门提起来也嵌不进去，怎么办呢？ 爷爷似乎驾轻就熟，上上下下看了看，就叫我把螺丝刀和锤子拿来。爷爷先用螺丝刀把固定木块的2颗成对角线的螺丝钉拧出来，然后用锤子一锤，木块就扁了，这样再把螺丝重新上上，门就能嵌进轨道了。 为了防止同类事件再次发生，爷爷给上下轨道和轮子都上了花生油，这样门拉起来就轻松多了，应该会持续一段长时间不会发生门因惯性和用力方向卡在轨道的事了。 经过这两次看爷爷修东西的经历，我学会了做事都要细心，要思考，会运用工具，这样才能达到目的。 Created by X&Y

2004年8月9日{初中时代的暑假日记}

2004年8月9日 晴 今天，我一口气看完了《杜立德医生》（Dr. Dolittle）。这本书我在四年级时已经看过，而今天重新看一遍，却领会到更多。 这本书讲的是喜爱动物又能和它们交谈的杜立德医生四处帮助各种各样动物的趣事。以前的我只觉得这本书有趣、滑稽；而今天的我，想到了爱心和童心，杜立德医生正是凭着一颗充满爱心、童心以及对科学热忱的心，才能学会各种动物的语言，只好它们的病，而且成为动物界的亲切和蔼又令人尊敬的好朋友。 看完这本书，我又想起前几天见到的两只猫。 一只是流浪猫。那天晚上我回家时，正好看见它趴在我们大院门口的垃圾袋旁，头探进垃圾袋里，一对前脚在垃圾袋里扒来扒去，找吃的。它是全白色的，很瘦，毛色不亮泽，脚掌附近的毛湿湿的，那天刚好下雨，想必是跑了许多路，最后饿了，便停在这里。我觉得它很可怜，想看看它的脸，便学猫“喵”了一声，因为我学猫叫是挺像的，所以我有信心这只猫肯定会看看我。谁知，“喵”过后，它猛地停住了前脚的扒动，抬起头，警惕地竖起耳朵，似乎全身的毛都竖起来了。我看见了它的双眼，虽然没有发光，但还是流露出迫人的寒气，我分不清那是恐惧，还是愤怒。妈妈也察觉了这只猫，叫我赶快走，要不它就追上来咬我了。我也有点害怕，因此就上楼了。 回到家后，我不停地跟妈妈说：“这只猫很可怜啊！”妈妈却说：“这只猫不好看，我不喜欢。”“你怎么可以这样说？”我十分惊讶。“这也不能怪我，要怪就怪它以前的主人不负责任，养一下子就把它扔了。”妈妈这句话把我给点醒了，是啊，究竟还是人类干的好事，不然，这只猫还在和它的伙伴游戏呢，怎么会沦落到在垃圾袋旁找吃的？ 另外一只是家猫，它就住在我们这栋楼的3楼。我上学时，几乎每个星期都有两三天的早晨，在3楼的楼梯道看见它趴在窗台上，我就“喵”一声和它打招呼，它也“喵”一声，似乎知道我在和它打招呼，它也和我打招呼。日子久了，我在3楼的楼梯道看见它，我“喵”一声，它也“喵”一声，直到我走到2楼的楼梯道，它听到我的声音，就把头往下探，我也在楼下望它，有时候甚至我走到楼底下都能看见它。这样一来，我在上学的路上就十分高兴了。 这只猫也是全白色的，不胖不瘦，挺健康，毛茸茸的，很干净，所以我每次看见它，都会很高兴。 也是一天晚上回家，楼梯道的灯坏了，黑乎乎的，我走在前面开路。刚走到3楼，就听见一只猫“喵”了两声，我抬头才发现它又蹲在窗台上，看着我。我兴奋极了，它主动和我打招呼！我赶紧也“喵”了几声。走到4楼，它往上看，我也往下望。进家门后，我欣喜地对妈妈说：“它认识我！它认识我了！”妈妈不信我，我解释道：“肯定是我以前经常和它打招呼，所以它认识我了，今天它主动和我打招呼了！”直到晚上睡觉，我还沉浸在快乐之中。 人与动物相处，可以是很融洽的。虽说学习动物的语言似乎不大可能，但是作者的这一想法不就是希望人类与动物能够融洽相处、相亲相爱吗？动物是人类的朋友，看见动物健康快乐，我们也会感到快乐，这个世界就能变得更美好！ Created by X&Y

《伊索寓言》读后感

《伊索寓言》读后感 《伊索寓言》是由短小精悍的寓言故事组成的，言语不多，却蕴含着深刻的道理。 看完书后仔细想了一下，发现有几个问题。 1、如钱钟书所作的伊索寓言新解，我认为《两只口袋》这样解释更好些：这个适用于不懂自我反省的人，这种人只看到别人的恶行，批评别人，却不知道自己也犯了同样的错误。 2、有些故事叙述的道理雷同。如《两个仇人》和《金枪鱼和海豚》，前者写的是“许多人只要能看见仇人先遭难，就不管自己的安危”，后者写的是“看见制造灾难的人遭殃，自己受到的灾难也就容易忍受了”；以及《劈橡树的人和橡树》和《鹰》，前者写的是“被自己人所害，比被外人所害，更叫人伤心”，后者写的是“为亲属所害是令人寒心的”我想，一般来说，创作的概念重复，并没什么意思，我不明白作者的用意；幸好这是在同一本书里，若不然，会令人感到有抄袭的味道。 3、有些故事叙述的道理太绝对，或者说是不符合现代文明。如《牧人和狼》所写“坏的天性养不成好的习惯”以及《打破神像的人》“尊敬坏人得不到一点好处，打了他，却能得到更多的利益”。先看前者，如果坏的天性的确养不成好的习惯，那么教育又有何用？上学的都经过筛选，都是天性“不坏”的吗？正因为有“坏”的天性，所以才需要教育者引导、帮助他，使他不要养成“不好”的习惯；若不然，犯罪事件必会大幅度上升。再看后者，打人无论在何时都是一种野蛮的行为，在现今社会，打坏人更是犯法的，这种行为值得提倡吗？ 以上便是我读完《伊索寓言》后想到的。 Created by X&Y

《爱的教育》读后感

《爱的教育》读后感 我在四年级时已经看过《爱的教育》这本书。它是我看的第一本外国名著，因此至今仍记忆犹新。 这本书里充满了爱：同学之间的友爱、父子的亲情、师生情、爱国之情；它勾画了一个十分美好的世界，人人相亲相爱，没有矛盾，没有阻隔，这也使四年级的我这样认为：只要多关心他人，真诚待人，别人也会同样待我，我应该努力去爱身边的每一个人，只要每个人都这样做，世界就会变得更美好。 但是随着时间流逝，我开始怀疑自己是不是错了？或许在书里我看到太多人类善良的一面，在现实中我却看见人类丑陋的面貌：自私自利、贪慕虚荣，抢着上公共汽车，插队，说粗口骂人，还有许多“小人”，这使我很失望，也使我认识到小说和现实的区别。 我终于明白了作者写这本小说的目的。意大利作家德·亚米契斯为9至13岁的孩子创作的这本小说，是让他们学会如何做人，如何爱别人，待他们长大后发现与现实有出入，便会以身作则，希望能够使世界充满爱。说是如此，但是在现实的压力下，能够做到真诚付出自己的爱的人有多少？ 举个最简单的例子：我曾多次因按部就班排队，得不到好东西，看着前面的人辛辛苦苦地挤上前，满意地看着自己努力的“成果”，我真不知道说什么好。 如此下去，理想的世界何时才会出现？说不定是几十甚至上百年后。人们的受教育程度不同，每个人的观点也不同，也不是人人都读过《爱的教育》，甚至有些读过这本书的人也不认同书中的观点，但我相信充满爱的世界一定比现在的世界好，起码人们能够相亲相爱，罪案、战争减少，从而生活素质就能提高；既然如此，我相信一定有些人会努力的爱身边的每一个人，他们的影响力可能不大，人类进步的脚步也不大，但他们都坚信，只要朝着这个目标努力奋斗，理想就一定能实现！ Created by X&Y
